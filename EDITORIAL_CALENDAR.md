## Content Creation Workflow

### 1. Research & Planning (30 min)
- Keyword research (Google Keyword Planner, Ahrefs)
- Competitor analysis
- Outline creation

### 2. Writing (2-3 hours)
- Follow template structure
- Include internal links
- Add Calendly CTA
- Optimize for keywords (natural integration)

### 3. SEO Optimization (30 min)
- Title 50-60 characters
- Meta description 150-160 characters
- H2/H3 hierarchy with keywords
- Image alt texts
- Internal/external linking

### 4. Visual Assets (30 min)
- Create cover image (1200x630px)
- Optimize images (WebP, <200KB)
- Add screenshots if applicable

### 5. Review & Publish (30 min)
- Spelling/grammar check
- Preview in dev mode
- Verify all links work
- Check mobile display
- Publish to main branch

**Total time per article**: 4-5 hours
